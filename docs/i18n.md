# D2-PromptSelector-comfyUI

[English](i18n_en.md) | [日本語](i18n.md) | [繁體中文](i18n_zh.md)

他言語対応ファイルに関する仕様。

## 対応言語

D2 Prompt Selector は標準で下記の言語に対応しています。

| 言語     | コード  |
| -------- | ------- |
| 日本語   | `ja`    |
| 英語     | `en`    |
| 簡体中文 | `zh`    |
| 繁体中文 | `zh-TW` |

ComfyUI の Settings > Comfy > Locale で選択した言語が反映されます。

## ユーザーによる言語追加

`D2-PromptSelector-comfyUI/web/locales/` フォルダに任意の言語コードで JSON ファイルを追加することで行えます。

例：韓国語 → `ko.json`

既存の JSON ファイルをコピーして翻訳してください。

翻訳テキストが記載されていない場合は英語が表示されます。

## 言語ファイルの構造

言語ファイルは「キー: 翻訳テキスト」を並べたフラットな JSON です。

```json
{
    "common.save": "保存",
    "button.edit": "編集",
    "search.placeholder": "検索...",
    "tag.delete.confirm.message": "「{name}」を削除しますか？"
}
```

| 部分                 | 説明                                                   |
| -------------------- | ------------------------------------------------------ |
| キー（左側 `"..."`） | アプリ内部で使う識別子。**変更・翻訳しないでください** |
| 翻訳テキスト（右側） | 画面に表示される文言。**ここだけを翻訳します**         |

### 編集のルール

- **キー（左側）は変更しない。** 右側の翻訳テキストだけを編集します。
- ファイルは **UTF-8** で保存し、JSON の文法（カンマ `,`・引用符 `"`・波括弧 `{}`）を壊さないようにします。
- 行末のカンマに注意してください（最後の項目にはカンマを付けない）。
- すべてのキーを翻訳する必要はありません。記載のないキーは英語 → 英語にもなければキー文字列の順で表示されます。

### 差し込み変数（プレースホルダ）

波括弧 `{ }` で囲まれた変数は、表示時に実際の値へ置き換わります。**翻訳テキスト内にそのまま残してください**（位置は文に合わせて移動可）。

| 変数              | 置き換わる内容           |
| ----------------- | ------------------------ |
| `{name}`          | 対象の表示名             |
| `{fileId}`        | ファイル名               |
| `{categoryCount}` | ファイル内のカテゴリ数   |
| `{itemCount}`     | ファイル内のプロンプト数 |

例：`"「{name}」を削除しますか？"` → 表示時に `「{name}」` がプロンプト名に置き換わります。

### HTML タグと改行

- 一部の値（`migration.message` / `file.delete.confirm.message` / `file.delete.confirm.typePrompt`）には `<br />`（改行）・`<strong>`（強調）・`<code>`（等幅）などの HTML タグが含まれます。**タグはそのまま残して**、間の文言だけ翻訳してください。
- `\n` は改行を表します（`category.delete.confirm.message` など）。そのまま残してください。

### 反映方法

ファイルを保存したあと、**ブラウザをリロード**すると翻訳が反映されます（ビルド不要）。表示する言語は ComfyUI の Settings > Comfy > Locale に従います。
