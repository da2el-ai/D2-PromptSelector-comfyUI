# D2-PromptSelector-comfyUI

[English](i18n_en.md) | [日本語](i18n.md) | [繁體中文](i18n_zh.md)

關於多語言對應檔案的規格。

## 支援的語言

D2 Prompt Selector 預設支援下列語言。

| 語言       | 代碼    |
| ---------- | ------- |
| 日文       | `ja`    |
| 英文       | `en`    |
| 簡體中文   | `zh`    |
| 繁體中文   | `zh-TW` |

會套用在 ComfyUI 的 Settings > Comfy > Locale 中選擇的語言。

## 使用者自行新增語言

只要在 `D2-PromptSelector-comfyUI/web/locales/` 資料夾中，以任意語言代碼新增 JSON 檔案即可。

範例：韓文 → `ko.json`

請複製現有的 JSON 檔案後進行翻譯。

未填寫翻譯文字的部分會顯示英文。

## 語言檔案的結構

語言檔案是由「鍵: 翻譯文字」並列而成的扁平 JSON。

```json
{
    "common.save": "儲存",
    "button.edit": "編輯",
    "search.placeholder": "搜尋...",
    "tag.delete.confirm.message": "要刪除「{name}」嗎？"
}
```

| 部分                  | 說明                                                       |
| --------------------- | ---------------------------------------------------------- |
| 鍵（左側 `"..."`）    | 應用程式內部使用的識別碼。**請勿變更或翻譯。**             |
| 翻譯文字（右側）      | 顯示於畫面上的文字。**只翻譯這個部分。**                   |

### 編輯規則

- **請勿變更鍵（左側）。** 只編輯右側的翻譯文字。
- 請以 **UTF-8** 儲存檔案，並注意不要破壞 JSON 語法（逗號 `,`、引號 `"`、大括號 `{}`）。
- 請留意行尾的逗號（最後一個項目不可加逗號）。
- 不需要翻譯所有的鍵。未填寫的鍵會以英文 →（英文亦無時）鍵字串本身的順序顯示。

### 插入變數（佔位符）

以大括號 `{ }` 包住的變數，會在顯示時替換為實際的值。**請在翻譯文字中原樣保留**（可依語句調整其位置）。

| 變數              | 替換內容                 |
| ----------------- | ------------------------ |
| `{name}`          | 對象的顯示名稱           |
| `{fileId}`        | 檔案名稱                 |
| `{categoryCount}` | 檔案內的分類數           |
| `{itemCount}`     | 檔案內的提示詞數         |

範例：`"要刪除「{name}」嗎？"` → 顯示時 `{name}` 會替換為提示詞名稱。

### HTML 標籤與換行

- 部分值（`migration.message` / `file.delete.confirm.message` / `file.delete.confirm.typePrompt`）含有 `<br />`（換行）、`<strong>`（強調）、`<code>`（等寬）等 HTML 標籤。**請原樣保留標籤**，只翻譯標籤之間的文字。
- `\n` 代表換行（例如 `category.delete.confirm.message`）。請原樣保留。

### 套用方式

儲存檔案後，**重新載入瀏覽器**即可套用翻譯（不需要建置）。顯示的語言會依照 ComfyUI 的 Settings > Comfy > Locale。
